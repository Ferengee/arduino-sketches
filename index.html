Arduino tiny core

For the arduino projects I build with an attiny I use the arduino-tiny core from http://code.google.com/p/arduino-tiny/

Trafic Lights

A sketch to control four (three led) traffic lights with an attiny85 programmed with arduino and used in a lego city.

I used to control leds with a tulip xt PC. But in these days one can easily program an attiny85 with an arduino uno as isp.

This traffic light controller controls 4 light posts with 3 led (red yellow and green) each. The twelve leds are charlieplexed. So I can use 4 digital pins on the attiny to drive the leds.

Sampled Button

A library for a button on a digital arduino pin. Compares the last (by default) three samples and calls event handlers on change and or pressed.

NightLight (An example from SampledButton)

An attiny85, three leds, a button and a ldr.

As soon as the device is powered on and the button is pressed, a light measurement is taken. The value of this measurement is used as a threshold to turn the light on or off.

A short click of the button selects the next led in line. A button press increases the intensity. If the led intensity is at is maximum the intensity is set to zero.
